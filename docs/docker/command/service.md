> 注意这部分是 Linux 的命令，如果是 Windows 有眼睛的都会启动和停止

- 启动 docker 服务

  ```sh
  systemctl start docker
  ```

  

- 停止 docker 服务

  ```sh
  systemctl stop docker
  ```

  

- 重启 docker 服务

  ```sh
  systemctl restart docker
  ```

  

- 查看 docker 服务状态

  ```sh
  systemctl status docker
  ```
