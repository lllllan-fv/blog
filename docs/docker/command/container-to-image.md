## 容器转为镜像

可以将容器转为镜像，并且会保留除了数据卷以外的其他数据。

```sh
docker commit <容器id> <镜像名>:<版本号>
```





## 镜像保存为压缩文件

但是镜像是不可以直接传输的，需要先保存为压缩文件

```sh
docker save -o <压缩文件名> <镜像名>:<版本号>
```





## 压缩文件恢复为镜像

将压缩文件进行传输之后可以由压缩文件恢复为镜像

```sh
docker load -i <y>
```
