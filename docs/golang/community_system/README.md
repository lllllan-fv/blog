## 项目内容

使用 Golang 开发的简单的聊天室。

项目地址：[community_system: Golang 练手项目翻新 (github.com)](https://github.com/lllllan-fv/community_system)



基本功能：

- 客户端连接上线
- 修改用户名
- 与其他客户端用户进行私聊
- 和所用在线用户进行公聊
