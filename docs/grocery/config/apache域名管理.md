> Apache 上管理不同的域名对应不同的项目



找到 `/apache/httpd.conf` ，在文件末添加你需要管理的域名

```xml-dtd
<VirtualHost *:80>
	DocumentRoot "*/data/www/default/project" 	# 项目的路径
	ServerName project.lllllan.cn 				# 域名
</VirtualHost>
```



**重启服务器**

```bash
service httpd restart
```
