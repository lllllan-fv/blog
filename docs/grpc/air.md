## 安装

https://github.com/cosmtrek/air

**注：请不要在有 (或父级目录有) `go.mod` 的目录执行该安装命令。**

```sh
go get github.com/cosmtrek/air
```



## 执行

之后运行 `air` 来替代原来的 `go run main.go`
